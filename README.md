# 📊 Superstore Sales Analysis Project

## 📌 Objective
Analyze sales and profit trends using Python and derive business insights.

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📂 Project Structure
- data/ → Raw dataset
- notebooks/ → Analysis notebook
- src/ → Data cleaning scripts
- images/ → Visualizations

## 📊 Key Analysis
- Sales by Category
- Profit by Region
- Correlation Analysis

## 📈 Insights
- Technology category has highest sales
- Some regions show lower profitability
- Discounts impact profit negatively
